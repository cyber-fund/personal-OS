# GTD

Living to-do list. Claude reads this at the start of most tasks and updates it as items move between sections. Keep lines short. One item per bullet. Move items, don't delete — active work flows `Now → Waiting / Next → Someday` or to `Done` at the bottom.

## Now

Focused work for today / this week. Keep this list small (≤7 items).

- 

## Next

Queued — pick up after `Now` clears.

- 

## Waiting

Blocked on someone else. Note who and since when.

- 

## Someday

Ideas, optional, no deadline.

- 

## Done

Recently completed items (last 7 days). Older items can be pruned.
