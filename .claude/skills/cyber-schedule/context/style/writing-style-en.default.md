# English Writing Style Guide

## Essay Structure

**Narrative mode:** Hook > Stakes > Mechanism > Turn (optional) > Landing

1. **Hook** — First sentence grabs attention. No throat-clearing. Jump into the insight.
2. **Stakes** — Why this matters. What happens if we get it wrong.
3. **Mechanism** — How it actually works. The insight, the data, the example.
4. **Turn** — Optional. A surprise, counterpoint, or reframe.
5. **Landing** — Forward momentum. What comes next. Never a summary.

## Length

- Short: 500-800 words
- Standard: 800-1500 words
- Deep: 1500-2500 words
- Stop when the idea is complete. Don't pad.

## Voice Rules

- Write like you're explaining to a smart friend
- Use concrete examples over abstractions
- Name names, cite numbers, reference real things
- One idea per paragraph
- Active voice. Subject-verb-object.
- If you can cut a word, cut it.

## Twitter/Short Form

- Lead with the insight, not the context
- One idea per tweet
- Threads: each tweet must stand alone
- No hashtags unless genuinely relevant
- Use numbers and specific claims over vague statements
